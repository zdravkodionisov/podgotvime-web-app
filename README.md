# Подготви-ме – уеб приложение

## Описание

„Подготви-ме“ е уеб приложение за подготовка по математика за НВО в 7. клас. Проектът е разработен с React JS и Vite.

Приложението позволява регистрация и вход на ученици, достъп до ученическо табло и стартиране на упражнения по избрани математически теми.

## Основни функционалности

- Начална landing страница
- Регистрация на ученик
- Вход в системата
- Ученическо табло
- Теми за подготовка
- Зареждане на задачи от backend
- Решаване на задачи
- Проверка на отговор
- Показване на резултат

## Използвани технологии

- React JS
- Vite
- React Router
- JavaScript
- CSS
- Fetch API
